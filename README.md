MarketPulse is a dashboard application for live markets, news, weather, and webcam feeds.

## Tech Stack

- **Client:** React, Vite, TypeScript, Tailwind CSS
- **API:** .NET (ASP.NET Core)

## Getting Started

### Prerequisites

- Node.js (for client)
- .NET SDK (for API)

### Run the client

```bash
cd client
npm install
npm run dev
```

### Run the API

Open the solution in Visual Studio or run from the `src/MarketPulse.API` directory.

### Build for production

```bash
cd client
npm run build
```

## Project Structure

- `client/` — React frontend (Vite)
- `src/` — .NET backend (API, Application, Domain, Infrastructure)
- `docs/` — Documentation and assets (e.g. logo image)
